# Cisco-ASA-Enterprise-Network-Security
The goal of this project is to design and manage security policies for an enterprise network by configuring a Network Firewall (Cisco ASA) and applying Access Control Lists to control inbound and outbound traffic.

## Overview
This project demonstrates enterprise network security using a Cisco ASA Firewall configured in Cisco Packet Tracer.

## Tools Used
- Cisco Packet Tracer
- Cisco ASA Firewall
- NAT / PAT
- Access Control Lists (ACLs)
- DNS

## Files Included
- Packet Tracer project file (.pkt)
- Detailed project report with screenshots
